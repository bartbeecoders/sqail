# Boilerplate

The "About sqail" block journalists paste at the bottom of an article. Two flavors.

## Standard (≈60 words)

**About sqail** — sqail is a free, open-source SQL editor for Windows, macOS, and Linux. Built on Tauri v2, it ships as a sub-20 MB native app with AI-assisted querying, multi-database support (PostgreSQL, MySQL, SQLite, SQL Server), and zero telemetry. Source code is on [Codeberg](https://codeberg.org/bartbeecoders/sqail) and [GitHub](https://github.com/bartbeecoders/sqail). Binaries are at [sqail.io](https://sqail.io).

## Minimal (≈25 words)

**About sqail** — a fast, open-source SQL editor with native AI integration. Cross-platform, under 20 MB, MIT licensed. Download at [sqail.io](https://sqail.io).
